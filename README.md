# The Muzic

> 프로그래머스 백엔드 데브코스 4기 개인미션 (김영주)

## 프로젝트 목적

- 개인이 인상 깊게 들은, 혹은 좋아하는 음악에 대한 정보를 아카이빙 하고 코멘트를 남길 수 있는 서비스를 제작한다.
- 지난 3주 간 학습한 SpringBoot 기본 CRUD 기능을 적용해본다.
- 프론트엔드와 백엔드를 분리하여 요청과 응답을 연습해본다.
- Spotify Web API를 이용하여 추가 기능을 구현해본다.
